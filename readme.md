# Student Management Console App

Консольное CRUD-приложение для управления списком студентов, разработанное на Java с использованием принципов объектно-ориентированного программирования. Позволяет добавлять, просматривать, изменять и удалять записи о студентах.

## Технологии

* Java 21
* Maven
* IntelliJ IDEA
* Git / GitHub

## Функциональность

* Добавление нового студента
* Просмотр списка всех студентов
* Поиск студента по ID
* Редактирование данных студента
* Удаление студента
* Валидация пользовательского ввода
* Обработка ошибок без завершения работы программы

## Структура проекта

* `model` — сущность Student
* `repository` — CRUD-операции и хранение данных
* `ui` — консольное меню и взаимодействие с пользователем
* `Main` — точка входа в приложение

## Запуск проекта

```bash
git clone <repository-url>
cd student-crud
mvn compile
```

После сборки запустите класс `Main.java`.

## Хранение данных

Данные хранятся в `ArrayList<Student>` в оперативной памяти и сбрасываются после завершения работы программы.
